# Custom Brand Logo

> ⚠️ **WARNING: DO NOT INSTALL THIS TO YOUR PRODUCTION!!!!!!!!!**

## Описание

Mu-plugin который заменяет стандартный WordPress логотип в левом верхнем углу админ-панели на ваш кастомный логотип бренда.

## Возможности

- ✅ Загрузка кастомного логотипа через админ-панель
- ✅ Отображение логотипа в левом верхнем углу админ-панели
- ✅ При клике на логотип открывается страница настроек
- ✅ Возможность удалить загруженный логотип
- ✅ Поддержка форматов: PNG, JPG, GIF, SVG

## Установка

1. Скопируйте файл `custom-brand-logo.php` в папку:
   ```
   wp-content/mu-plugins/
   ```

2. Админ-панель автоматически создаст новый пункт меню "Настройки темы"

## Как использовать

### Загрузка логотипа

1. Перейдите в **левом меню** → **Настройки темы**
2. На странице настроек нажмите **"Выбрать файл"**
3. Загрузите изображение логотипа (PNG, JPG, GIF или SVG)
4. Нажмите кнопку **"Загрузить логотип"**
5. Логотип появится в левом верхнем углу админ-панели

### Удаление логотипа

1. Перейдите в **Настройки темы**
2. Найдите раздел **"Текущий логотип"**
3. Нажмите кнопку **"Удалить логотип"**

## Рекомендации

### Размер логотипа

- **Минимум:** 16x16px
- **Оптимально:** 32x32px или 48x48px
- **Максимум:** зависит от вашего желания

### Формат файла

| Формат | Описание |
|--------|---------|
| **PNG** | Лучше всего с прозрачностью |
| **SVG** | Идеален, масштабируется идеально |
| **JPG** | Подойдет если нет прозрачности |
| **GIF** | Поддерживается |

### Цвет логотипа

- Логотип отображается на **тёмном фоне** админ-бара
- Используйте **белый**, **светлый** или **яркий** цвет
- Это обеспечит хорошую видимость

## Как работает

```
WordPress Logo (стандартный)
         ↓
   [МУ-ПЛАГИН]
         ↓
Custom Brand Logo (ваш логотип)
```

Когда логотип загружен:
1. Плагин удаляет стандартный WordPress логотип
2. Добавляет ваш кастомный логотип на его место
3. При клике открывает страницу настроек темы

## Технические детали

- Логотип сохраняется как обычное вложение в медиа-библиотеку WordPress
- При удалении логотипа из библиотеки опция также автоматически очищается
- Логотип масштабируется до высоты 20px в админ-баре
- Использует стандартные WordPress функции загрузки файлов

## Отключение

Чтобы отключить плагин:

1. Удалите или переименуйте файл `custom-brand-logo.php` в папке `wp-content/mu-plugins/`
2. Вернётся стандартный WordPress логотип
3. Загруженные логотипы останутся в медиа-библиотеке

## Частые вопросы

### Логотип не отображается?

- Проверьте что файл загружен правильно
- Убедитесь что расширение файла: PNG, JPG, GIF или SVG
- Перезагрузите страницу админ-панели (Ctrl+Shift+R)

### Страница настроек не открывается?

- Убедитесь что у вас есть права администратора
- Проверьте что плагин активирован (файл находится в `wp-content/mu-plugins/`)

### При клике логотип не открывает страницу?

- Найдите пункт **"Настройки темы"** в левом меню админ-панели
- Это основной способ доступа к настройкам логотипа
